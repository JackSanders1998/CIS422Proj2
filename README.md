## Installation:

### For Mac users:

Download [XCode] (https://developer.apple.com/xcode/)  
After downloading XCode, open terminal and run:  
`sudo gem install cocoapods`

Then: 
`git pull "https://github.com/JackSanders1998/CIS422Proj2/tree/main/Fetch" `
